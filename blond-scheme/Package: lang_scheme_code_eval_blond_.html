BLOND: Non-reflective simulator of a reflective tower.

lang/scheme/code/eval/blond/
This directory contains the non-reflective simulator of a reflective tower described in Olivier Danvy and Karoline Malmkjaer, "Intensions and Extensions in a Reflective Tower", in Proceedings of the 1988 ACM Conference on Lisp and Functional Programming, pages 327-341, 1988. The interpreter is fully meta-circular.
