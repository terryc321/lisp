\subsection{the BAD-OR macro}

Let us define a two argument BAD-OR macro in terms of LET and IF

\begin{verbatim}
(defmacro bad-or (a b) 
  (let ((tmp a))
     (if tmp tmp b)))
\end{verbatim}

Now let us put the macro to work in an environment where TMP is bound

\begin{verbatim}
(let ((tmp 5)) 
  (bad-or #f tmp))
\end{verbatim}

\begin{verbatim}
=> (let ((tmp 5)) 
    (let ((tmp #f))
     (if tmp tmp tmp)))
\end{verbatim}

\begin{verbatim}
=>  (let ((tmp #f))
      (if tmp tmp tmp))
\end{verbatim}

\begin{verbatim}
=>  (if #f #f #f)
\end{verbatim}

\begin{verbatim}
=>  #f
\end{verbatim}
